# NES Desktop Emulator

Uses the [`jc-nes`](../jc-nes/) crate and the [SDL crate](https://crates.io/crates/sdl2) to build a desktop emulator.

## Building

```
$ cargo build --release
```

## Running

```
$ cargo run --release
```

## Controls

| Keyboard   | Gamepad       |
| ---------- | ------------- |
| A          | A             |
| B          | B             |
| Z          | Start         |
| X          | Select        |
| Arrow Keys | Arrow Buttons |
