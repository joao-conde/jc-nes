# Desktop NES emulator

Utilizes the [`jc-nes`](../../jc-nes/) crate and the [SDL crate](https://github.com/Rust-SDL2/rust-sdl2) to build a desktop application.

## Running

```
$ cargo run --release
```

Drag and drop your ROM to play.

## Player Controls

| Keyboard   | Gamepad       |
| ---------- | ------------- |
| A          | A             |
| B          | B             |
| Z          | Start         |
| X          | Select        |
| Arrow Keys | Arrow Buttons |
